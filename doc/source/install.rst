Installation Instructions
=========================


Python
#######

An Anaconda version of python3 is recommended. DCT currently requires,

* Numpy
* SciPy
* Networkx
* CVXPy
* matplotlib
* json

CVXPY
######

First ecos should be installed using::

	>>> conda install ecos

from an Anaconda command prompt.

Then::

	>>> pip(3) install cvxpy

DCT
###

Hopefully::

	>>> pip(3) install DCT
